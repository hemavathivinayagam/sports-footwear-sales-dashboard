# sports-footwear-sales-dashboard
## Project Overview
This project is an interactive Power BI dashboard designed to analyze sales performance, customer behavior, and product demand in the Sports Footwear industry.

The dashboard provides insights into:
- Revenue performance
- Units sold
- Discount analysis
- Customer ratings by brand
- Country-wise sales
- Gender & category trends
- Top demanded models
- Payment method distribution

## Business Problem
The company operates across multiple countries, brands, and product categories.  
Management needs a centralized dashboard to:

- Identify top-performing brands
- Understand customer satisfaction
- Track high-demand models
- Monitor discount impact
- Analyze gender-based buying patterns

This dashboard helps convert raw data into actionable business insights.
